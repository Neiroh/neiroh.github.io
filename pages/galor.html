Galor Project

Galor Project was a little project we got to do one month before ending the theorical part of the course.

In this case I can share you my code via Github, as Yumeanime. You can see it in this repository: Galor Repository

The website consists in an art gallery where you can Like and comment the pictures you want. This page was done with native PHP, native CSS, native JS and HTML5 and AJAX. We got to use an API, and I decided to use Pexels.

In Mobile you can scroll in X-axis to watch more pictures
