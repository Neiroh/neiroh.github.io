Fallon Carrington Project

This website was, as I said, my end of superior grade project. This took me two months and a half, and it's my favourite one

In this case I can share you my code via Github. You can see it in this repository: Yumeanime Repository

This page was about showing some Anime news to the viewers and to research about all the animes the API storages. It had a few description of two or three, even four paragraphs talking about the anime topic, a title and an image.

This page was done with native PHP, native CSS, native JS and HTML5. I had to use an API, and I decided to use API Jikan Moe. Firstly, the API I selected (because it was a free project and we could do whatever we wanted) was API Aniapi but the day before I got to present my project, the API stopped working and I had to change it the last night
